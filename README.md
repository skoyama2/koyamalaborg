# Koyama Lab Website

Static website for Koyama Lab @ MGB & Broad & HMS.

## Files

- `index.html`: main lab website
- `pi.html`: principal investigator profile
- `styles.css`: shared site styling
- `script.js`: mobile navigation behavior
- `assets/`: site images and member photos

## Deployment

This site does not require a build step. For Netlify, set the publish directory to `.` and leave the build command blank. For GitHub Pages, publish from the repository root.
